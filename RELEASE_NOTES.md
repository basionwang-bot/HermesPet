## Hermes 桌宠 v1.4.6

### 继续收口崩溃 + 修好上一版的位置偏移

接着 v1.4.5 的启动崩修复，这版把**同一族**剩下的崩溃风险一次性收口，并修好了 v1.4.5 带出来的一个位置偏移。

**这版做了什么**

- **继续收口同族崩溃**：在较新的 macOS（26.5 及以上）上，打开「舰队 / 工作流面板 / 博物馆 / AI 网页 / 竞技场 / 头像裁剪」这些功能时**可能闪退**——跟上一版的启动崩是同一个根因（窗口承载方式在新系统的屏幕刷新机制下会触发致命异常）。这版把这些窗口**全部换成更稳的实现**，点开不再崩。
- **修好 v1.4.5 的位置偏移**：上一版把几个窗口换成更稳的实现时漏了一行设置，导致**悬停灵动岛弹出的系统卡片**、以及**工作台里的内容**位置往下偏了一截。现已回正。
- **一批小修复**：
  - 工作台多标签页的状态不再被别处的 AI 活动（舰队 / 工作流 / 早报等）干扰。
  - 英文界面下某个深度工作流不再卡住反复重试。
  - 会议纪要的诊断信息不再被误报成「错误」。
  - 屏幕自动回复在目标聊天窗已关闭时，不再误报「已回复」（避免消息石沉大海）。

> 说明：这些崩溃只在 macOS 26.5+ 上出现，开发机是 26.3 复现不出来，所以靠成熟的修复范式收口。给受影响的朋友带来的麻烦非常抱歉。

### 下载安装

按你的 Mac 芯片选对应的版本：

- **Apple Silicon**（M1 / M2 / M3 / M4 系列）→ 下载 `HermesPet-1.4.6-AppleSilicon.dmg`
- **Intel 芯片** → 下载 `HermesPet-1.4.6-Intel.dmg`
- 不确定？点屏幕左上角  菜单 →「关于本机」，看「芯片」那一行。

安装步骤：

1. 打开下载好的 DMG
2. 把「Hermes 桌宠」拖进「应用程序」文件夹
3. 从启动台 / Spotlight 打开「Hermes 桌宠」即可

老用户无需手动操作，App 会在 24 小时内自动检测到新版本并提示更新（也可手动：设置 → 检查更新）。
